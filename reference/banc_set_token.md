# Set the token to be used to authenticate to banc autosegmentation resources

Set the token to be used to authenticate to banc autosegmentation
resources

## Usage

``` r
banc_set_token(token = NULL)
```

## Arguments

- token:

  An optional token string. When missing you are prompted to generate a
  new token via your browser.

## Value

The path to the token file (invisibly)
