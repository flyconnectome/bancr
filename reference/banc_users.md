# User information (name + CAVE ID) for active BANC users

The purpose of this table is to map CAVE users IDs to names, in order to
credit annotation work done in BANC CAVE. This information is based on
[google
sheet](https://docs.google.com/spreadsheets/d/1UFmeWr2uF9jTLVMw3bD6nM3ejM-b-HDZz6sQBPTEoZ8/edit?gid=1163959922#gid=1163959922).

## Usage

``` r
banc_users
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 228
rows and 3 columns.
